# Brain-Trainer
Brain Trainer Final College Project

## What's this project ?

Brain Trainer is a game that's designed for ages 8-12 to challenge the abilities,
and make practice and studies more fun with 2 different games (Puzzle, Math)

## What the goal of this project ?

The goal of this project is develop your thinking abilities and train the brain to learn faster and easier through digital games.
It's designed to help people that prefer to learn digitaly, lack of focus or just for different way of learning.
